01How it works

One architecture, three stages. The same RA + EEGNet + IM-TTA pipeline runs for both within- and cross-subject evaluation.

Stage 1

Covariance alignment

Whiten each recording by its mean spatial covariance — Euclidean (arithmetic mean) or Riemannian/centroid (SPD geometric mean). From signals alone, no labels.

→
Stage 2

EEGNet backbone

UnifiedEEGNet, a compact EEGNet-style CNN (~34k params). The depthwise spatial conv behaves like CSP filters in the whitened space.

→
Stage 3

IM-TTA novel

At inference, adapt BatchNorm to the target subject's unlabelled trials by information maximization — sharpen predictions while keeping the class mix balanced.

03Architecture & math

The backbone is a small EEGNet variant; the alignment and adaptation steps are where the gains come from.

Alignment

Whiten each recording by a covariance reference, applied as X′ = M·X.

EA  M = R−1/2,   R = meann( XnXnT / T ) # arithmetic mean cov
RA  M = invsqrtm( 𝔊({covn}) ) # 𝔊 = SPD geometric (Fréchet) mean

RA respects the curved geometry of the SPD manifold and is better conditioned when covariances are heterogeneous (e.g. across subjects).

IM-TTA loss

Adapt only BatchNorm affine params on the target's unlabelled trials.

L = 𝔼[ H(p) ] − H( 𝔼[p] )
  • 𝔼[H(p)] ↓ — sharpen each prediction (confidence)
  • H(𝔼[p]) ↑ — keep the batch class mix balanced (anti-collapse)

Dropping the second term reduces to plain Tent, which collapses on hard subjects — see the ablation below.

Within-subject negatives

Tried, did not beat RA alignment — kept on record.

  • Band-narrowing (4–40 / 8–30 Hz) — worse than broadband
  • Augmentation expansion ×2–3 — worse
  • Refit on train+val — worse
  • Learnable / per-trial spatial alignment — neutral→worse
  • Dual-geometry (EA+RA) fusion — worse (overfits)

Every run's per-subject CSV and training log is archived in the repo under results/.

07Training setup

Optimisation

  • AdamW · OneCycleLR · label smoothing 0.1
  • MixUp + signal-space augmentation (S&R, freq-mask, channel-drop, jitter)
  • Checkpoint & early-stop on the validation split
  • Cross-subject supervised pretrain → per-subject fine-tune (within); from scratch (LOSO)

IM-TTA at inference

  • Adapts BatchNorm affine params only (rest frozen)
  • BN uses batch statistics; ~5 entropy/diversity steps
  • No labels, no ensembling — a single forward model
  • Results reported over multiple seeds (mean ± std)

08Key takeaways

  • Alignment is the foundation. Whitening covariance (EA or RA) lifts every setting; RA edges EA on the cross-subject side.
  • Test-time adaptation is a cross-subject lever. IM-TTA adds ~+3 points on LOSO for both datasets, where the subject gap is largest.
  • The diversity term is essential. Plain entropy-minimisation collapses on hard subjects and gives no gain; balancing the class mix is what unlocks it.
  • One pipeline, two protocols, two datasets. The same RA + EEGNet + IM-TTA recipe is competitive within-subject and improves cross-subject — no per-setting redesign.

09Reproduce

MOABB pulls the datasets automatically. One command runs both protocols.

uv sync
python main.py --dataset iv2a --protocol both --align ra --tta_steps 5 --tta_div 1.0 --seed 42
python main.py --dataset iv2b --protocol both --align ra --tta_steps 5 --tta_div 1.0 --seed 42

Flags: --dataset {iv2a,iv2b} · --protocol {within,loso,both} · --align {none,ea,ra} · --tta_steps N · --tta_div {1 = info-max, 0 = entropy-only} · --seed.
