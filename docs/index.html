BCI Competition IV · Motor Imagery

Aligned EEGNet + IM-TTA

A compact pipeline for motor-imagery EEG: align each recording's covariance, run it through a small EEGNet-style network, then adapt the model to the target subject at inference. Two datasets, within- and cross-subject.

01How it works

One architecture, three stages. The same RA + EEGNet + IM-TTA pipeline is used for both within- and cross-subject evaluation.

Stage 1

Covariance alignment

Whiten each recording by its mean spatial covariance — Euclidean (arithmetic mean) or Riemannian/centroid (SPD geometric mean). Computed from the signals alone.

→
Stage 2

EEGNet backbone

UnifiedEEGNet — a compact EEGNet-style CNN (~34k params). Its depthwise spatial conv acts like CSP filters in the whitened space.

→
Stage 3

IM-TTA novel

At inference, adapt BatchNorm to the target subject's unlabelled trials by information maximization — sharpen predictions while keeping the class mix balanced.

Datasets: BCI IV-2a (BNCI2014-001, 4-class, 22 ch) and IV-2b (BNCI2014-004, 2-class, 3 ch), 9 subjects each. Within-subject uses the official session split; LOSO trains on eight subjects and tests on the ninth. A validation split picks the checkpoint; the test set is scored once.

Within-subject negatives

Things that did not beat RA alignment — kept for the record.

  • Band-narrowing (4–40 / 8–30 Hz) — worse than broadband
  • Augmentation expansion ×2–3 — worse
  • Refit on train+val — worse
  • Learnable / per-trial spatial alignment — neutral to worse
  • Dual-geometry (EA+RA) fusion — worse (overfits)

Every run's per-subject CSV and training log is archived in the repo under results/.

04Reproduce

MOABB pulls the datasets automatically. One command runs both protocols.

uv sync
python main.py --dataset iv2a --protocol both --align ra --tta_steps 5 --tta_div 1.0 --seed 42
python main.py --dataset iv2b --protocol both --align ra --tta_steps 5 --tta_div 1.0 --seed 42

Flags: --dataset {iv2a,iv2b} · --protocol {within,loso,both} · --align {none,ea,ra} · --tta_steps N · --tta_div {1 = info-max, 0 = entropy-only} · --seed.
